# Hackathon Starter Guide

この Skill は完成版ではありません。
AGIラボハッカソン参加者が、`誰のどんな仕事を、どこまで自律化するのか` を
最初に整理するためのスターターです。

## 使い方

この Skill が呼ばれたら、以下の順でユーザーを手伝う。

1. 作りたいものを 1 行で言い切る
2. 誰のためのものかを決める
3. どんな入力で起動するかを決める
4. どんな結果が出るかを決める
5. どこが「自律性」なのかを言葉にする
6. README に書くべき要点を 5 行でまとめる
7. 3 分デモ動画の流れを箇条書きにする

## 最初に聞くこと

- 誰のどんな面倒を減らしたいですか
- どんなきっかけで動きますか
- 最後に何が出てくると嬉しいですか
- 途中で人が介入しなくてよい部分はどこですか

## 出力フォーマット

以下の形で短く整理する。

```text
1. 一言でいうと:
2. 対象ユーザー:
3. 入力:
4. 出力:
5. 自律的なポイント:
6. README の骨子:
7. デモ動画の流れ:
```

## ルール

- 難しい設計から入らず、まずは 1 つの仕事を終わらせる形に絞る
- 「便利そう」ではなく「何を任せられるか」が伝わる言葉にする
- README とデモ動画の骨子まで考えて、他の人が試せる形を意識する
- ユーザーがまだ曖昧なら、3 案まで具体例を出して選べるようにする
